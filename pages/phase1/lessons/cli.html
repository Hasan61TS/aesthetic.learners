Command Line Basics

Essential Commands

The command line is your primary interface for interacting with Linux. Let's learn some basic commands:

ls List files and directories
cd Change directory
pwd Print working directory
Try These Commands
user@linux:~$

Quick Check 1

Which command shows your current directory location?

ls
pwd
cd

Directory Navigation

Moving around the file system is a crucial skill. Here are important navigation commands:

cd / Go to root directory
cd ~ Go to home directory
cd .. Go up one directory

üí° Navigation Analogy

Think of directories like rooms in a building. cd is like walking through doors, pwd tells you which room you're in, and ls shows you what's in the room.

File Operations

Let's learn how to manipulate files and directories:

mkdir Create a new directory
touch Create a new empty file
rm Remove a file
cp Copy files or directories
