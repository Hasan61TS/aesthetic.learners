Understanding Paths

There are two ways to specify file locations in Linux:

/home/user Absolute path (starts from root)
./Documents Relative path (from current location)
../ Parent directory

Advanced Navigation Techniques

Let's explore some advanced navigation commands and shortcuts:

cd - Go to previous directory
cd ~ Go to home directory
cd ../../ Go up two directories

Practice Exercise

Try this navigation sequence:

  1. Go to root directory (cd /)
  2. Navigate to your home (cd home/user)
  3. Go up one level (cd ..)
  4. Return to previous directory (cd -)
