Introduction to Linux

What is Linux?

Linux is an open-source operating system kernel that forms the foundation of many different operating systems (distributions).

Think of Linux as the engine of a car. Just like how different car manufacturers can use the same type of engine to build different cars, different organizations can use Linux to build different operating systems.

Linux Distributions

Linux distributions (or "distros") are complete operating systems built around the Linux kernel. Popular distributions include:

  • Ubuntu - User-friendly and popular for beginners
  • Kali Linux - Specialized for security testing and hacking
  • Fedora - Advanced features and cutting-edge technology
  • Debian - Known for stability and security

üí° Simple Analogy

If Linux is an engine, distributions are complete cars. Each car (distro) uses the same basic engine (Linux kernel) but adds different features and designs to serve different purposes.

The Terminal: Your Primary Tool

The terminal (or command line interface) is your main way of interacting with Linux. Let's try some basic commands:

Terminal
user@linux:~$

Try these commands:

whoami - Shows your username pwd - Shows current directory ls - Lists files and folders
