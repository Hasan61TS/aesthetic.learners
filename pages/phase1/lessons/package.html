Package Management

Understanding Package Management

Package management is how Linux systems install, update, and remove software. The most common package manager in Debian-based systems (like Ubuntu and Kali) is APT (Advanced Package Tool).

üí° Simple Analogy

Think of a package manager like an app store for your computer. Just like how you install apps on your phone through the app store, you install software on Linux through the package manager.

Essential APT Commands

Here are the most important APT commands you'll use:

apt update Update package list
apt install Install new package
apt remove Remove package
apt upgrade Upgrade all packages
apt search Search for packages

Software Repositories

Repositories are servers that hold collections of packages. Linux systems know which repositories to use through the sources list.

sources.list Repository configuration file
add-apt-repository Add new repository
